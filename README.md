## Usage

### Controls

|      Key       |            Action             |
| :------------: | :---------------------------: |
| `Up`/`Down`    | Move along X-axis             |
| `Left`/`Right` | Move along Z-axis             |
| `Space`        | Increase drop speed           |
| `S`, `D`, `F`  | Rotate about X, Y, and Z axes |
| `Q`            | Pause the game                |
| `W`            | Toggle music                  |
